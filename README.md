# WebDAV Clip Ext

通过 WebDAV 在设备间同步文本和文件的浏览器扩展。

## 功能

- 📝 文本同步 - 快速保存和复制文本片段
- 📁 文件传输 - 上传/下载文件，支持拖拽
- 🔍 搜索 - 快速查找历史记录
- 👁️ 预览 - 支持图片和文本文件预览
- 🔒 安全 - 密码加密存储，HTTP 连接警告

## 安装

1. 下载或克隆本仓库
2. 打开浏览器扩展管理页面
   - Edge: `edge://extensions/`
   - Chrome: `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择项目文件夹

## 使用

1. 点击扩展图标打开面板
2. 进入设置，填写 WebDAV 服务器信息
3. 开始同步文本和文件

## 要求

- 支持 WebDAV 协议的服务器（如 Nextcloud、坚果云等）
- Edge 或 Chrome 浏览器
